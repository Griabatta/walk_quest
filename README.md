# Вечерний квест

Личный PWA-квест для вечерних прогулок. Версия использует **Яндекс Карты JavaScript API** и строит **пешие маршруты** через `ymaps.route(..., { routingMode: 'pedestrian' })`.

## Что умеет

- определяет геолокацию телефона;
- принимает цель в шагах;
- переводит шаги в примерную длину маршрута;
- генерирует несколько случайных точек вокруг текущего местоположения;
- строит к каждой точке пеший маршрут через Яндекс Карты;
- выбирает маршрут, который ближе всего к заданной цели;
- показывает маршрут на карте;
- открывает маршрут во внешних Яндекс Картах;
- засчитывает победу, когда пользователь входит в радиус 50/75/100 м;
- показывает экран победы, конфетти, звук, вибрацию где поддерживается;
- хранит историю прогулок в localStorage;
- устанавливается на Android/iPhone как PWA без Google Play / App Store.

## Как запустить

Геолокация и PWA нормально работают только на HTTPS.

Самые простые варианты:

1. Загрузить папку проекта на GitHub Pages, Netlify, Vercel или Cloudflare Pages.
2. Открыть ссылку на телефоне.
3. Разрешить геолокацию.
4. На Android в Chrome нажать «Установить» / «Добавить на главный экран».
5. На iPhone открыть в Safari → «Поделиться» → «На экран “Домой”».

## Важное про ключ Яндекс Карт

В проект уже вставлен ключ JavaScript API:

```text
bd809e5e-0ef7-406a-848f-e83d08542653
```

Лучше в кабинете Яндекса ограничить ключ доменом, на котором будет лежать приложение. Если ключ оставить без ограничений, его сможет скопировать кто угодно из исходного кода страницы.

## Где менять ключ

В файле `index.html`:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ&lang=ru_RU"></script>
```

## Ограничения MVP

- Реальные шаги не считаются: введённые шаги используются как цель для длины маршрута.
- Подбор маршрута зависит от доступности Яндекс Карт и лимитов ключа.
- В фоне iPhone/Android могут реже обновлять геолокацию, поэтому во время прогулки лучше держать приложение открытым.
- Если Яндекс не смог построить маршрут к случайным точкам, приложение выберет запасную точку без точной длины маршрута.
