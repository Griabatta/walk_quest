# Вечернее дело — чистая версия PWA

Это новая аккуратная сборка без фоновых картинок, стикеров и декоративной мешанины. Сохранена только игровая механика и начинка приложения.

## Что внутри

- PWA для Android и iPhone без публикации в сторах
- Яндекс Карты JavaScript API
- подбор случайной цели под заданное число шагов
- построение пешеходного маршрута
- фильтрация очевидных водных точек через геокодинг
- радиус выполнения миссии 50/75/100 м
- детективные миссии
- экран победы
- случайная песня как финальная задача
- история прогулок

## Дизайн

Интерфейс собран заново: спокойная светлая палитра, чистые карточки, понятная типографика, без наложения текста и лишних картинок.

## Установка

1. Загрузи папку на HTTPS-хостинг: Netlify, Vercel, GitHub Pages, Cloudflare Pages.
2. Открой ссылку на телефоне.
3. Разреши геолокацию.
4. На Android: Chrome → Установить приложение / Добавить на главный экран.
5. На iPhone: Safari → Поделиться → На экран «Домой».

Если раньше стояла старая версия, удали старую иконку PWA и добавь приложение заново, чтобы обновился кэш.
