# Вечернее дело — PWA-квест прогулок

Версия дизайна **по загруженным фонам**: стартовый экран, экран миссии и экран победы собраны ближе к макету с растровыми фонами, золотыми рамками, зелёными кнопками, бумажными карточками и рисованными следами.

## Что работает

- PWA для Android и iPhone без публикации в магазинах
- Яндекс Карты JavaScript API с указанным ключом
- подбор пешего маршрута под количество шагов
- проверка случайных целей через геокодинг, чтобы не отправлять в очевидную воду
- детективные миссии
- экран победы с финальной песней
- история прогулок

## Как установить

1. Загрузить папку на HTTPS-хостинг: Netlify, Vercel, GitHub Pages, Cloudflare Pages и т.п.
2. Открыть сайт на телефоне.
3. Разрешить геолокацию.
4. Android: в Chrome нажать «Установить приложение» / «Добавить на главный экран».
5. iPhone: открыть в Safari → «Поделиться» → «На экран Домой».

## Если не обновился дизайн

Из-за service worker телефон может держать старую версию. Тогда:

1. открой сайт и обнови страницу 2–3 раза;
2. удали старую иконку PWA с экрана;
3. добавь приложение на экран заново.
