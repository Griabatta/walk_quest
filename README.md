# Вечернее дело — PWA-квест прогулок

Обновлённая версия в более **растровом, атмосферном детективном стиле**.

## Что есть в этой сборке

- PWA для Android и iPhone (через Add to Home Screen / На экран «Домой»)
- Яндекс Карты JavaScript API
- подбор пешего маршрута под заданное число шагов
- защита от очевидных водных точек через геокодинг
- детективные миссии
- экран победы с финальной песней
- история прогулок
- новый дизайн: закатный маленький город, тёплые огоньки, бумажные карточки, более «растровое» ощущение интерфейса

## Установка

1. Загрузить папку на HTTPS-хостинг (Netlify, Vercel, GitHub Pages и т.п.)
2. Открыть сайт на телефоне
3. Разрешить геолокацию
4. При желании установить как PWA на главный экран

## Важно

Если на устройстве уже стояла старая версия, лучше:

1. открыть новый сайт,
2. обновить страницу пару раз,
3. удалить старую иконку PWA,
4. поставить заново.

Это нужно из-за service worker и кэша.
